📸 METHOD 1: SCREENSHOT (सबसे आसान - 5 minutes)
═══════════════════════════════════════════════════════════════

यह सबसे simple method है - कोई software install नहीं करना!

STEP-BY-STEP:

1. SOCIAL PREVIEW IMAGE बनाएं:
────────────────────────────────────────────────────────────
   a) social-preview-template.html को browser में खोलें
   b) F11 दबाएं (full screen)
   c) Zoom = 100% set करें (Ctrl+0)
   d) Windows: Win+Shift+S (Snipping Tool)
      Mac: Cmd+Shift+4
   e) Pure 1200x630 area select करें
   f) Save as: social-preview.jpg
   g) Done! ✅

2. ICON IMAGE बनाएं:
────────────────────────────────────────────────────────────
   a) icon-template.html को browser में खोलें
   b) F11 दबाएं (full screen)
   c) Screenshot लें (512x512 area)
   d) Save as: icon-512.png
   e) Done! ✅

3. RESIZE करें (Different Sizes):
────────────────────────────────────────────────────────────
   Online tool use करें:
   
   • https://imageresizer.com
   • https://iloveimg.com/resize-image
   
   icon-512.png से बनाएं:
   □ icon-192.png (192x192)
   □ icon-144.png (144x144)
   □ icon-72.png (72x72)
   □ favicon-32x32.png (32x32)
   □ favicon-16x16.png (16x16)
   □ apple-touch-icon.png (180x180)

🎨 METHOD 2: CANVA (Best Quality - 10 minutes)
═══════════════════════════════════════════════════════════════

Professional quality के लिए Canva use करें (Free!)

STEP 1: CANVA SETUP
────────────────────────────────────────────────────────────
1. Go to: https://canva.com
2. Sign up (free account)
3. Start designing

STEP 2: SOCIAL PREVIEW (1200x630)
────────────────────────────────────────────────────────────

CREATE NEW:
1. Click "Custom Size"
2. Width: 1200px, Height: 630px
3. Click "Create new design"

DESIGN:
1. Background:
   • Add gradient background
   • Colors: #e8a020 (golden) to #c8900a
   
2. Add Text:
   • Main: "गहोई बंधु" (88px, Bold, White)
   • Sub: "GAHOI BANDHU" (48px, Bold, Dark Blue #1a3a5c)
   • Tagline: "गौतम बुद्ध नगर का सबसे बड़ा गहोई नेटवर्क"
   
3. Add Feature Boxes:
   • 3 boxes with icons
   • Box 1: 👥 Directory | Member Network
   • Box 2: 💑 Matrimony | Rishta Service
   • Box 3: 🎉 Community | Events & Updates
   
4. Add Bottom Badge:
   • "Join 500+ Families • Greater Noida"
   
5. Download:
   • File type: JPG
   • Quality: Best
   • Save as: social-preview.jpg

STEP 3: ICON (512x512)
────────────────────────────────────────────────────────────

CREATE NEW:
1. Custom Size: 512x512px
2. Square canvas

DESIGN:
1. Background:
   • Gradient circle
   • Golden color (#e8a020)
   
2. Add Text:
   • "GB" (140px, Bold, Navy #1a3a5c)
   • "गहोई" below (48px, White)
   
3. Add decorative elements:
   • Circular border
   • Corner decorations
   
4. Download:
   • File type: PNG
   • Transparent: No
   • Save as: icon-512.png

STEP 4: RESIZE IN CANVA
────────────────────────────────────────────────────────────
1. Open icon-512.png design
2. Click "Resize" button
3. Enter each size:
   • 192x192 → Save as icon-192.png
   • 180x180 → Save as apple-touch-icon.png
   • 32x32 → Save as favicon-32x32.png
   • 16x16 → Save as favicon-16x16.png
